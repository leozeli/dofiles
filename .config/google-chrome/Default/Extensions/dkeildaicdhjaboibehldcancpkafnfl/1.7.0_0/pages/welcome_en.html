Welcome

This is a bookmark extension. You can find the "Card Bookmark" icon in the extended toolbar in the upper right corner. After clicking, your bookmark will be displayed in a pop-up window.

Compared with the bookmark bar of the browser, it does not occupy the webpage display space, and can be used at any time.
