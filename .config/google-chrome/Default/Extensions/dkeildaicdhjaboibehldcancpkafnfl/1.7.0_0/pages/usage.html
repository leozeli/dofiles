使用说明

使用快捷键

1. 在 Chrome 上可以使用 Ctrl+B 打开卡片书签（其他浏览器可能会有冲突），你也可以打开 chrome://extensions/shortcuts 找到卡片书签进行修改；

2. 打开卡片书签后可以使用 Tab 键选择书签， Enter 打开；

3. 在书签上点击右键后可以使用 Shift 键区间选中多个书签；

其他 Tips

搜索书签时，点击搜索结果的图标可以打开其所在的书签夹；

如果需要连续打开多个书签，请在设置中开启 连续打开；
